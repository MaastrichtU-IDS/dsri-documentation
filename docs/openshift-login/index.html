Login to OpenShift

You will first need to authenticate to the DSRI cluster using the OpenShift client.

Get the command from the Copy Login Command button, in the user details, at the top right of the OpenShift webpage (your token is automatically provided).

oc login https://openshift_cluster:8443 --token=MY_TOKEN

Paste the command in your terminal.
