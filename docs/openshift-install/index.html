Installation

Install the OpenShift client

On Linux

wget https://github.com/openshift/origin/releases/download/v3.11.0/openshift-origin-client-tools-v3.11.0-0cbc58b-linux-64bit.tar.gz
tar xvf openshift-origin-client-tools*.tar.gz
cd openshift-origin-client*/
sudo mv oc kubectl /usr/local/bin/

See the official documentation on Linux for more details.

On Mac

brew install openshift-cli

For people that know what they are doing: download the .zip from GitHub Releases and unzip it in your path.

See the official documentation on MacOS for more details.

On Windows

Follow the official documentation on Windows to install the oc client.

Install the Helm client

This feature has not been tested at the moment. Please contact us if you are interested in deploying Helm Charts.

Helm is a popular package manager for Kubernetes. It allows you to easily deploy Helm Charts built by the community.

On Linux

curl https://get.helm.sh/helm-v3.0.0-linux-amd64.tar.gz > helm-v3.0.0-linux-amd64.tar.gz
tar -zxvf helm-v3.0.0-linux-amd64.tar.gz
sudo mv linux-amd64/helm /usr/local/bin/helm

See Helm documentation for Linux.

On MacOS

brew install helm

See Helm documentation for MacOS.

On Windows

Install using Chocolatey.

choco install kubernetes-helm

See Helm documentation for Windows.
