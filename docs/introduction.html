Introduction

The Data Science Research Infrastructure is an OpenShift cluster based on the MapR Distributed FileSystem, using Kubernetes containers orchestration.

The DSRI provides a graphical user interface to easily run and manage services (OKD). Multiple workflow orchestration tools can be deployed, such as Argo workflows, Nextflow or CWL Calrissian.

Documentation to deploy Helm Charts is also available.

See the DSRI workshop presentation for more details.
