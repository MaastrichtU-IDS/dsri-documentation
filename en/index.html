Run jobs in parallel on multiple nodes

OpenShift and Kubernetes allow you to run any Docker container in parallel on multiple nodes.

OpenShift also enables you to leverage the power of MapReduce jobs.

Data stored on the MapR FileSystem are replicated over multiple nodes for high availability and fault tolerance.
